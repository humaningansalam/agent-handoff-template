# Tasks

This folder contains live task files, parent tasks, and completed child tasks that remain reopenable until their parent task is completed.
Use `project/BOARD.md` as the task board and entrypoint.
Start most tasks from `TEMPLATE.md`; use `PARENT_TEMPLATE.md` only for coordinating parent tasks.
Standalone completed tasks move to `project/archive/tasks/` and are removed from `project/BOARD.md`.
Completed child tasks are also removed from `project/BOARD.md`, but may stay here until their parent task is completed.
